# Parametro P0612 - Viviendas que utilizan leña o carbón para cocinar, que disponen de estufa o fogón con chimenea

Archivo de mineria de datos. Parametro: P0612 - Viviendas que utilizan leña o carbón para cocinar, que disponen de estufa o fogón con chimenea

**Contenido**: Estimadores de las viviendas particulares habitadas en las que sus ocupantes utilizan leña o carbón para cocinar y su distribución porcentual según disponibilidad de estufa o fogón con chimenea por municipio
